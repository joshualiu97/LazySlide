# LazySlide-App
IdeaHacks 2020
Melody Chen, Ellie Cheng, Joshua Liu, Ryan Tong, Eric Wang

LazySlide IOS/Android App developed with React Native.
Opens curtains with the press of a button.
Uses socket.io to connect with server and send request to Raspberry Pi.

Work in progress:
Be able to know which state the curtain is in(open or closed).
